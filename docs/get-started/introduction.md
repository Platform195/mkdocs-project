# Introduction

Uplift Hub helps you manage advertising and marketing activity end to end —
from planning and activation through to post-campaign and year-on-year
reporting. This guide walks through the essential workflows: setting up
accounts, creating campaigns, adding line items, uploading creatives and
tracking performance.

You'll also learn about [adding creatives](../creatives/add.md), applying
[custom targeting](../line-items/targeting.md), and using our
[reporting tools](../reporting/tracking.md). A
[glossary](../reference/glossary.md) at the end clarifies key terms like
packages, ad creatives and pacing.
