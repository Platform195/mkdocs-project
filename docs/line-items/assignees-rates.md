# Assignees &amp; product rates

You can assign each line item to a specific stakeholder for different stages of
its lifecycle. The assignee is emailed every time the line item's status
changes. Assignees can be set at the product level or overridden per line item.

Select the **Pencil** icon on a line item row to edit its assignee, partner
rate or fulfilment cost (when there's a third-party).
