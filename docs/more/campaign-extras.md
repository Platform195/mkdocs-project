# Campaign extras

Add additional campaign details on the campaign's
<span class="ui-ref">Brief</span> tab — including
<span class="ui-ref">Target audience</span>,
<span class="ui-ref">Partner focus</span> and
<span class="ui-ref">Partner KPIs</span>.

You can also set a campaign to <span class="pill paused">Paused</span> or
<span class="pill archived">Archived</span> at any time. Both states
automatically suspend delivery of any on-site digital activity.
