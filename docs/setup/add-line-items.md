# Adding line items

If you selected a **Package** type campaign, line items are generated
automatically. If you selected **Custom**, you'll add line items individually
to match the media plan.

## Two ways to add a line item

1. **From inside a campaign** — open a campaign, switch to the
   <span class="ui-ref">Line items</span> tab and click
   <span class="ui-ref">Add new line item</span>.
2. **From the Line Items page** — select <span class="ui-ref">Line Items</span>
   in the sidebar, click <span class="ui-ref">Add new line item</span>, and
   assign it to a campaign from the dropdown.

## Add details

Provide as much information as possible. Use <span class="ui-ref">Notes</span>
for special instructions (targeting, prioritisation) and include links to the
media plan or brief.

!!! tip "Need help choosing a product?"
    Leave a comment on the campaign's <span class="ui-ref">Comments</span> tab.
    Type ++at++ followed by a teammate or your Platform 195 representative to
    notify them.
