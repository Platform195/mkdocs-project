# Creating a campaign

A **Campaign** represents all the components of a sold order — usually mirroring
a contract, purchase or insertion order with a client.

## Steps

1. Select <span class="ui-ref">Campaigns</span> from the sidebar, then click
   <span class="ui-ref">Add campaign</span>.
2. Fill in campaign details. Set a <span class="ui-ref">Sales probability</span>
   (100 = fully sold/closed) if useful, then click
   <span class="ui-ref">Save</span>.
3. Edit later as needed — every field except <span class="ui-ref">Type</span>
   and <span class="ui-ref">Sales probability</span> can be changed from the
   <span class="ui-ref">Basic details</span> tab.

!!! note "Campaign type"
    Choose **Package** to auto-populate line items from a predefined product
    bundle, or **Custom** to build line items individually.
