# Go-live workflow

Once all assets are uploaded and approved per the campaign and advertiser
requirements, set the line item's status:

- <span class="pill ready">Ready Live</span> — if the start date hasn't been reached yet.
- <span class="pill live">Live</span> — if the campaign should start immediately.

!!! tip "Auto-activation"
    Any line item set to <span class="pill ready">Ready Live</span> activates
    automatically the moment its start date arrives — no manual flip required.
