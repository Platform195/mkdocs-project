title: Campaign extras
description: Brief details, paused/archived states, and other campaign-level controls.
---

import { Ui } from "/snippets/ui.mdx";
import { Pill } from "/snippets/pill.mdx";

Add additional campaign details on the campaign's <Ui>Brief</Ui> tab — including
<Ui>Target audience</Ui>, <Ui>Partner focus</Ui> and <Ui>Partner KPIs</Ui>.

You can also set a campaign to <Pill tone="paused">Paused</Pill> or
<Pill tone="archived">Archived</Pill> at any time. Both states automatically
suspend delivery of any on-site digital activity.
